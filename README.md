# Introduction
Working in progress, this project is a graph-vector database and GRAG pipeline with a load balance backend.

Tech stack:
- Python
- LangChain
- HuggingFace
- Qdrant
- Neo4j (planning)
- Docker Compose (planning)
- FastAPI (planning)
- Celery (planning)
- RabbitMQ (planning)
- Redis (planning)
- React (planning)
